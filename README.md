# Lodestone

A canonical operator methodology for working with AI systems.

## What this is

Lodestone names the practice that keeps work true to its frame. It is one of four reference artifacts in the Clarethium body:

- **Touchstone** validates work against quality standards.
- **Whetstone** sharpens craft.
- **Lodestone** orients practice. *(this artifact)*
- **FieldReceipts** captures proofs.

Lodestone defines the stance, the calibration moves, the failure shapes, and the practice loop that compose disciplined work with AI systems.

This is a reference document, not a tutorial. Read it as you would read a field manual or a published standard: a citable canonical text for those ready to adopt the practice.

## Who this is for

Senior operators whose work with AI systems is shaped continuously through the operator-AI loop. Engineers, researchers, designers, writers, analysts. The audience is anyone whose output quality depends on loop quality.
